# 🤖 NexusTG | Твой цифровой ИИ-двойник для Telegram

[![GitHub Repo](https://img.shields.io/badge/GitHub-Repository-blue?logo=github)](https://github.com/newfpv/NexusTGTest)

**NexusTG** — это мощный Telegram-юзербот, которым вы управляете через удобного классического бота с кнопками. Вы настраиваете его один раз, и он начинает работать в фоновом режиме на вашем аккаунте.

Это не просто скрипт, это ваш **цифровой двойник**. Он умеет общаться за вас с помощью нейросетей, читать удаленные сообщения собеседников, переводить голосовые в текст и многое другое!

---

## 🌟 Что умеет бот? (Модули)

* 🧠 **AI Twin (ИИ Двойник)** — Умный автоответчик на базе Google Gemini. Имитирует ваш стиль общения, "печатает" текст с задержками (как живой человек) и даже имеет "режим сна", когда он не трогает сообщения.
* 🕵️ **Spy-Module (Шпион / Сохранялка)** — Скрытно сохраняет **удаленные** и **измененные** сообщения собеседников, а также скачивает "исчезающие" (одноразовые) фото и видео в ваш секретный чат-дамп.
* 🎙 **Голос в текст** — Автоматически (или по вашей команде) переводит голосовые сообщения и "кружочки" в текст. Если ГС длинное — нейросеть сделает краткую выжимку (саммари).
* 🎭 **Фейк активность** — Позволяет включить у собеседника статус, что вы "печатаете", "записываете видео" или "играете в игру" на заданное время.
* 🧠 **Ручной ИИ (команда `.ai`)** — Вы можете прямо в любом диалоге написать `.ai помоги решить задачу`, и бот прочитает контекст переписки и выдаст ответ от нейросети прямо от вашего лица.
* 🛒 **Список покупок** — Умный парсер. Вы скидываете текст в чат (например: "купи хлеб, 2 литра молока и сыр"), а бот превращает это в красивый кликабельный чек-лист.
* 👤 **Инфо-модуль** — Просмотр подробной скрытой информации о любом пользователе Telegram.

## 🛠 ПОДРОБНАЯ ИНСТРУКЦИЯ ПО УСТАНОВКЕ (Для новичков)

Не пугайтесь! Процесс установки выглядит объемным, но если идти строго по шагам, вы справитесь за 10 минут.

### Шаг 1: Подготовка (Получаем нужные ключи)

Перед тем как запускать программу, нам нужно получить 3 вещи:

1.  **Токен бота (Управление):**
    * Зайдите в Telegram и найдите бота [@BotFather](https://t.me/BotFather).
    * Напишите ему команду `/newbot`.
    * Придумайте имя (любое) и юзернейм (обязательно заканчивается на `bot`, например `MyNexus_bot`).
    * BotFather пришлет вам **Токен** (длинная строка вида `1234567890:AAH...`). Скопируйте и сохраните её.
2.  **API ключи Telegram (Для юзербота):**
    * Перейдите на официальный сайт [my.telegram.org](https://my.telegram.org) и войдите по своему номеру телефона.
    * Перейдите в раздел **API development tools**.
    * Заполните форму (названия можно писать любые на английском) и нажмите "Create application".
    * Скопируйте оттуда две вещи: **API_ID** (цифры) и **API_HASH** (длинная строка из букв и цифр).
3.  **Ключ нейросети Gemini:**
    * Перейдите на сайт [Google AI Studio](https://aistudio.google.com/app/apikey) (нужен Google-аккаунт; если сайт не открывается — включите VPN).
    * Нажмите слева **Get API key** -> **Create API key** -> **Create API key in new project**.
    * Скопируйте полученный ключ.

### Шаг 2: Установка программы

У вас есть два пути: простой (через Docker) и ручной (на Windows). Выберите один из них.

#### Вариант А: Установка на Windows (Вручную)
1.  **Установите Python:** Скачайте [Python 3.11 или новее](https://www.python.org/downloads/). **ОЧЕНЬ ВАЖНО:** при установке поставьте галочку внизу окна — **"Add Python to PATH"**!
2.  **Скачайте проект:** Нажмите зеленую кнопку **Code -> Download ZIP** на этой странице GitHub и распакуйте папку в удобное место.
3. **Создайте и настройте конфигурационный файл `.env`**
   Поскольку на серверах нет привычного «Блокнота» с мышкой, мы всё будем делать специальными командами. Не пугайтесь, просто повторяйте:
   
   * **Шаг 3.1: Переименование файла.** В папке проекта уже лежит шаблон-пустышка. Нам нужно переименовать его из `.env.example` в рабочий файл `.env`. Для этого введите команду:
     `mv .env.example .env`
     *(и нажмите Enter)*
   
   * **Шаг 3.2: Открытие редактора.** Теперь откроем этот файл в самом простом консольном текстовом редакторе, который называется `nano`. Введите:
     `nano .env`
     *(нажмите Enter — интерфейс консоли немного изменится, вы окажетесь внутри текстового файла)*
   
   * **Шаг 3.3: Вставка токена.** 1. Используйте **стрелочки на клавиатуре** (вверх/вниз/влево/вправо), чтобы спустить мигающий курсор к строчке, которая начинается с `TG_BOT_TOKEN=`.
     2. Убедитесь, что после знака `=` нет пробела.
     3. **Вставьте ваш токен от BotFather.** *⚠️ ВАЖНО: Привычная комбинация `Ctrl+V` в консоли обычно не работает! Чтобы вставить текст, скопированный на вашем компьютере, просто кликните внутри черного окна **правой кнопкой мыши** (в программах вроде PuTTY) или нажмите комбинацию `Shift + Insert`.*
   
   * **Шаг 3.4: Сохранение и выход.** Когда вы вставили токен (получится что-то вроде `TG_BOT_TOKEN=1234567890:AAH_ваш_токен`), файл нужно сохранить:
     1. Нажмите на клавиатуре одновременно **Ctrl + O** (английская буква О, от слова Output). Это команда сохранения.
     2. В самом низу экрана появится надпись *«File Name to Write: .env»* (Имя файла для записи). Просто нажмите **Enter**, чтобы подтвердить.
     3. Теперь нажмите **Ctrl + X**, чтобы закрыть редактор `nano` и вернуться в обычную командную строку.
4.  **Установите зависимости и запустите:**
    * В папке с проектом кликните правой кнопкой мыши по пустому месту (в Windows 11 нажмите "Показать дополнительные параметры") и выберите **"Открыть в Terminal"** (или Открыть PowerShell).
    * В черном окне по очереди введите две команды (после каждой жмите Enter):
        `pip install .`
        *(Дождитесь окончания загрузки библиотек)*
        `python main.py`
    * Если вы не закрыли черное окно — бот работает!

#### Вариант Б: Установка через Docker (Для серверов и продвинутых)
Если вы знаете, что такое Docker, то просто:
1.  Склонируйте репозиторий: `git clone https://github.com/newfpv/NexusTGTest.git`
2.  Перейдите в папку: `cd NexusTGTest`
3.  Создайте файл `.env` (на базе `.env.example`) и впишите туда `TG_BOT_TOKEN`.
4.  Запустите контейнер: `docker-compose up -d --build`

### Шаг 3: Настройка в самом Telegram

Самое сложное позади! Теперь вся настройка происходит прямо в интерфейсе Telegram.

1.  Перейдите в бота, которого вы создали в BotFather.
2.  Нажмите кнопку **СТАРТ** (`/start`).
3.  Бот поприветствует вас и попросит по очереди отправить то, что мы добыли на Шаге 1:
    * Ваш `API_ID` (отправьте цифрами).
    * Ваш `API_HASH`.
    * Ваш ключ от `Gemini`.
4.  Далее бот предложит **"Войти в юзербота"**.
5.  Введите свой номер телефона от Telegram (в формате `+79991234567`).
6.  Telegram пришлет вам код подтверждения прямо в приложение. Введите этот код с помощью **кнопок на экране** в боте.
7.  Если у вас стоит облачный пароль (2FA) — бот попросит его ввести. Это безопасно, пароль никуда не передается.

🎉 **ГОТОВО!** Вы увидите главное меню.

## ⚙️ Как пользоваться?

* Перейдите в **Главное меню** вашего управляющего бота.
* **"Мои чаты"** — здесь вы можете выбрать любой диалог и настроить ИИ-двойника конкретно под этого человека (например, чтобы бот отвечал начальнику серьезно, а друзьям — мемами).
* **"Настройки ядра"** — здесь можно менять ключи API, менять часовой пояс и настраивать глобальное поведение ИИ.
* Для работы "Списка покупок" или ручного ИИ, просто зайдите в нужный чат (с вашего основного аккаунта) и напишите триггер-команду (по умолчанию `.ai ваш запрос` или `.shop хлеб, молоко`).

### Ограничения (Для стабильной работы)
* Бот требует постоянной работы скрипта. Если вы закроете консоль на компьютере (или выключите компьютер) — бот уснет. Для работы 24/7 рекомендуется устанавливать бота на дешевый VDS-сервер (например, на Ubuntu).
* Для корректного извлечения данных с YouTube может потребоваться файл `cookies.txt` (можно загрузить в настройках бота).
